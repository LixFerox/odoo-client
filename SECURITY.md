# Security Policy

## Supported Versions

Security fixes are provided for the latest published `1.x` release line.

## Reporting a Vulnerability

Please do not report security vulnerabilities in public issues.

Instead:

1. Open a private security advisory in GitHub for this repository, if available.
2. If advisories are not available, contact the maintainer privately.

Please include:

- A clear description of the issue
- Impact and affected components
- Reproduction steps or proof of concept
- Suggested mitigation (if known)

You can expect an initial response within 7 days.
